## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Icons:** Font Awesome 6
- **Animations:** Intersection Observer API (Scroll Reveal)
- **Contact System:** FormSubmit.co (AJAX/Fetch)
- **Hosting:** Vercel

## ✨ Key Features
- 💻 **Terminal Loader:** A developer-themed boot sequence with animated typing and progress tracking.
- 🌓 **Dark/Light Mode:** Seamless theme switching with local storage persistence.
- 📱 **Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
- 📧 **Functional Contact Form:** Real-time email delivery with AJAX status feedback.
- 🚀 **Performant:** Lightweight assets and smooth scroll reveal animations.

## ⚙️ Installation & Local Setup
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dipraj2/dipraj-portfolio.git
   ```
2. **Navigate to the folder:**
   ```bash
   cd dipraj-portfolio
   ```
3. **Open in Browser:**
   Simply open `index.html` or use a Live Server extension in VS Code.

## 🚀 Deployment Instructions
This project is optimized for **Vercel**:
1. Connect your GitHub account to Vercel.
2. Select the `dipraj-portfolio` repository.
3. Click **Deploy**. Vercel will automatically detect the static files.
